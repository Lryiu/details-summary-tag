What are details and summary tags?

The details tag specifies additional details that the user can open and close on demand. The details tag is often used to create an interactive widget that the user can open and close. By default, the widget is closed. When open, it expands, and displays the content within. Any sort of content can be put inside the details tag. More information

The summary tag defines a visible heading for the details element. The heading can be clicked to view/hide the details. More information
